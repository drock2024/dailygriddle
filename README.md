# Daily Griddle

## Game Description
Daily Griddle is a daily trivia game built to your personal favorites. Tell the game what series you like and it will generate you both a "wordle" style and "grid" style game where you test your knowledge on characters from that series.

## General Rules
Once a guess has been made, no updates to your favorites selection will reflect until the following day.
Prior to making a guess, you can change your favorites selection and they will update real time.

## "Wordle" Rules
Each day a character from one of your favorite series will be chosen as the answer of the day.
You have 6 guesses to get the correct answer, each guess will populate a row and tell you about their characteristics.
A green square means your inputted character matches the correct character in that category.
A red square means your inputted character does not match the correct character in that category.
A yellow square means your inputted character does not match the correct character but it will show you whether the inputted category value is higher or lower than the correct character's.
A gray square means either the inputted character or the correct character has an unknown or n/a value in that category.

## "Grid" Rules
A 3x3 grid is generated each day with different category buckets along the rows and columns.
For each square in the grid, you must input a character that fits both the row and column categories.
To win the game you must fill out the whole square, characters may be used multiple times.
You have 12 total guesses to fill the board.
